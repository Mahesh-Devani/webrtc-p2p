      How Nostr Connect works

      1. Share your public key (above) with your contact via any messenger
      2. Paste their public key and click Connect
      3. Both peers must be online — the app uses Nostr relays to exchange encrypted signaling
      4. Once connected, all communication is direct peer-to-peer (WebRTC)

      Tokens use the format: P2P2-<base64url_data>
      
            
      To decode manually:
      
            
        
              
      1. Remove the P2P2- prefix
        2. 
              
      2. Convert base64url → base64 (replace - with +, _ with
                /, pad with =)
              
        3. 
              
      3. Base64-decode to get compressed bytes
        4. 
              
      4. Inflate/decompress using pako (zlib deflate)
        5. 
              
      5. The result is a JSON object with: { v, t, ts, s, c }
        6. 
            
      
            
      Or run in browser console:
      
            
      // Paste your token:
const token = 'P2P2-...';
const b64 = token.slice(5).replace(/-/g,'+').replace(/_/g,'/');
const bin = atob(b64.padEnd(b64.length + (4 - b64.length % 4) % 4, '='));
const bytes = new Uint8Array([...bin].map(c=>c.charCodeAt(0)));
const json = new TextDecoder().decode(pako.inflate(bytes));
console.log(JSON.parse(json));
      
            
      No data is sent to any server — everything is decoded locally in your
                browser.
